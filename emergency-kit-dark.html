Kit de Emergência

Sistema de gerenciamento de itens essenciais para situações de emergência, com controle de estoque, alertas de validade e checklist personalizado.

Sobre o Projeto

O Kit de Emergência é uma aplicação web desenvolvida para auxiliar no gerenciamento de itens essenciais em situações de emergência. O sistema permite o controle detalhado do estoque, com alertas de validade, quantidade mínima e checklist personalizado para verificações periódicas.

Funcionalidades Principais

  • Gestão de itens por categoria
  • Controle de validade
  • Alertas de estoque baixo
  • Checklist personalizado
  • Relatórios exportáveis
